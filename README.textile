h3. My dotfiles for OS X and Linux.

To install:
* git clone git://github.com/cdmwebs/dotfiles.git
* cd dotfiles
* rake install

To update plugins after installation:
* cd plugin-dir
* git remote update

*If you do fork, make sure to change your gitconfig to match your name and email address!*
